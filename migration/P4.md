## P4-03 创建 route/index.php

**目标文件**：`route/index.php`（新建）

**来源**：`application/route.php`，提取前台路由段（约 80%）

**先查看 route.php 完整内容**：
```bash
cat application/route.php
```

**TP5 → TP8 路由转换规则**：

```php
<?php
use think\facade\Route;

// 1. 全局 pattern（原 __pattern__ 键）
Route::pattern([
    'id'    => '[\s\S]*?',
    'page'  => '\d+',
    'sid'   => '\d+',
    'nid'   => '\d+',
]);

// 2. 普通路由（TP5 数组格式 → TP8 链式）
// TP5: 'sitehome' => 'index/home/index'
Route::get('sitehome', 'index/home/index');

// TP5: 'vod-<id>-<sid>-<nid>-<page>.html' => ['index/vod/detail', ...]
Route::get('vod-<id>-<sid>-<nid>-<page>.html', 'index/vod/detail');

// 3. 包含方法限制的路由
// TP5: ['index/user/login', ['method' => 'post']]
Route::post('user/login', 'index/user/login');
```

**注意**：
- TP5 的 `<id>` 占位符在 TP8 中仍然有效（`<name>` 格式）
- TP5 数组第三个元素（pattern 覆盖）在 TP8 用 `->pattern(['id' => '\d+'])` 追加

## P4-04 Console Commands 配置迁移

**目标文件**：`config/console.php`（新建）

**来源**：`application/command.php`（TP5 命令注册文件）

**查看现有命令列表**：
```bash
cat application/command.php 2>/dev/null || ls application/command/
```

已知 10 个命令：AdminResetPassword / CacheFlush / DbExport / DbImport / DbSearchReplace / Info / SeoAiGenerate / SiteDestroy / SiteInstall / Tune

**TP8 config/console.php**：
```php
<?php
return [
    'commands' => [
        'admin:reset-password' => \app\command\AdminResetPassword::class,
        'cache:flush'          => \app\command\CacheFlush::class,
        'db:export'            => \app\command\DbExport::class,
        'db:import'            => \app\command\DbImport::class,
        'db:search-replace'    => \app\command\DbSearchReplace::class,
        'info'                 => \app\command\Info::class,
        'seo:ai-generate'      => \app\command\SeoAiGenerate::class,
        'site:destroy'         => \app\command\SiteDestroy::class,
        'site:install'         => \app\command\SiteInstall::class,
        'tune'                 => \app\command\Tune::class,
    ],
];
```

**注意**：命令名（键）从 TP5 的 `getName()` 返回值读取，先检查各 Command 类：
```bash
grep -rn "getName\|protected \$name" application/command/ | head -15
```

**命令文件 use 声明更新**：
```bash
find application/command/ -name "*.php" -exec sed -i \
  -e 's/use think\\Db;/use think\\facade\\Db;/g' \
  -e 's/use think\\Cache;/use think\\facade\\Cache;/g' \
  {} +
```

TP8 的 `think\console\Command` 命名空间不变，`Input`/`Output` 接口不变，大多数命令无需额外改动。
